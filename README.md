# Employee Management System v2

A modern v2 of the original C++ employee management console app, rebuilt as a deploy-ready React dashboard.

## Features

- Animated glass hero section
- Add, edit and delete employee records
- Live search by ID, name, role, department or status
- Employee status tracking
- Active employee, department and payroll summaries
- JSON export for records
- React + TypeScript + Tailwind CSS
- shadcn-style `/src/components/ui` structure

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This project is ready for Vercel, Netlify or GitHub Pages. Build output is generated in `dist`.

## Project Story

v1 was a C++ console-based employee management system with file storage and CRUD operations. v2 keeps the same core idea but presents it as a polished web dashboard suitable for a GitHub portfolio.
