
# 👨‍💼 Employee Management System V2

A modern, beautiful, and deploy-ready **Employee Management System V2** built with **React, TypeScript, Tailwind CSS, and Vite**.

This is the upgraded version of my original **Employee Management System V1**, which I built around **7 months ago** as a C++ console-based project.  
V2 transforms the same idea into a polished web dashboard with a professional UI, better user experience, and deployment-ready structure.

---

## 🚀 Live Project Purpose

Employee Management System V2 is designed to help manage employee records through a clean and responsive dashboard.

It includes employee CRUD operations, real-time search, payroll summaries, status tracking, and JSON export functionality.

---

## ✨ Features

- 🎨 Modern animated hero section
- 👥 Add new employee records
- ✏️ Edit existing employee details
- 🗑️ Delete employee records
- 🔍 Live employee search
- 🏢 Department tracking
- ✅ Employee status management
  - Active
  - On Leave
  - Inactive
- 💰 Monthly payroll summary
- 📊 Dashboard statistics
  - Active employees
  - Total departments
  - Total payroll
- 📥 Export employee data as a `.json` file
- 📱 Fully responsive design
- ⚡ Fast performance with Vite
- 🧩 Clean component-based React structure
- 🎯 Ready to deploy on Vercel, Render, or Netlify

---

## 📥 JSON Export Feature

This project includes an **Export** button that downloads the current employee records as a JSON file.

The exported file is named:

```bash
employees-v2.json
```

This makes it easy to save, backup, or reuse employee data outside the application.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/swastikjha008-jpg/Employee-Management-System-V2.git
```

Go to the project folder:

```bash
cd Employee-Management-System-V2
```

Install dependencies:

```bash
npm install
```

Run the project locally:

```bash
npm run dev
```

---

## 🏗️ Build for Production

```bash
npm run build
```

The production-ready files will be generated inside the:

```bash
dist
```

folder.

---

## 🌐 Deployment

You can deploy this project easily on:

- Vercel
- Render
- Netlify

### Render Static Site Settings

Use these settings:

```bash
Build Command: npm install && npm run build
Publish Directory: dist
Branch: main
```

Root Directory should be left empty.
